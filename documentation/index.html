1. Project Setup

Welcome to the Mobintix Infotech template. This is a clean, modern, and responsive HTML5/CSS3 template designed for IT consulting firms and digital agencies.

Prerequisites

Since this is a static template, you don't need any complex backend environment. You only need:

  • A text editor (VS Code, Sublime Text, etc.)
  • A modern web browser (Chrome, Firefox, Edge, Safari)

How to Run

  1. Unzip the downloaded folder.
  2. Right-click on index.html and select Open with > Google Chrome.
  3. For a better experience, we recommend using the Live Server extension in VS Code.

4. Customization

Changing Colors

We use CSS Variables for easy color management. Open style.css and look for the :root section at the top.

:root {
    --primary: #064e3b;      /* Main Brand Color */
    --secondary: #10b981;    /* Hover/Accent Color */
    --accent: #d1fae5;       /* Light Backgrounds */
    ...
}

Simply change these hex codes to match your brand identity.

Changing Fonts

Mobintix Infotech uses Google Fonts (Outfit & Inter). To change them:

  1. Go to Google Fonts and select your desired font.
  2. Update the <link> tag in index.html (head section).
  3. Update the font variables in style.css: 
    --font-heading: 'YourFont', sans-serif;

5. Changing Images

All images used in the template are placeholders. Replace them by dropping new images into the assets/ folder and updating the src attributes in the HTML.

Tip: For best results, use compressed, WebP, or optimized JPG images.

6. Contact Form

The contact form is frontend-only by default. To make it functional, you will need a backend endpoint or a service like Formspree.

Example using Formspree:

<form action="https://formspree.io/f/yourformid" method="POST">
    ...
</form>
