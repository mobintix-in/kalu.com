1. Project Setup

This is a high-performance, static HTML5/CSS3 template designed for modern IT businesses options. No complex build tools are required to get started.

Prerequisites

  • A code editor (VS Code, Sublime Text, etc.)
  • A modern web browser (Chrome, Firefox, Edge, Safari)

Installation

Since no build process is strictly necessary, you can run the files directly:

  1. Unzip the Infotech-HTML-Template.zip file.
  2. Right-click on index.html in the HTML folder.
  3. Select Open with > Your Preferred Browser.
Pro Tip: For the best development experience, use the "Live Server" extension in VS Code. It auto-reloads the page as you save changes.

4. Dark Mode Setup

The template features a robust dark mode implementation that respects user preference and saves state.

How it Works

  • Auto-save: The script uses localStorage to remember if the user prefers 'dark' or 'light' theme.
  • Toggle: The moon/sun icon in the navbar toggles the .dark-mode class on the <body> tag.

Customizing Colors

Dark mode colors are defined separately in style.css. To modify them, find this block:

body.dark-mode {
    --primary: #57d16b;       /* Neon Green Accent */
    --light: #121212;         /* Dark Background */
    --bg-surface: #1e1e1e;    /* Card Backgrounds */
    --header-bg: rgba(18, 18, 18, 0.95);
}

5. Customization

Global Colors

The template uses CSS Custom Properties (Variables) for global color management. Edit the :root selector at the top of style.css:

:root {
    --primary: #0b2e13;      /* Core Brand Color (Dark Green) */
    --secondary: #2ea043;    /* Primary Action Color (Emerald) */
    --accent: #57d16b;       /* Highlights */
    --light: #f4fbf6;        /* Page Background */
}

Fonts

We use Outfit for headings and Inter for body text via Google Fonts.

  1. Select your new font on fonts.google.com.
  2. Replace the <link> tag in the <head> of your HTML files.
  3. Update the CSS variables: 
    --font-heading: 'YourFont', sans-serif;

6. Changing Images

Replace existing placeholder images in the assets/ folder with your own. Ensure you keep similar filenames (e.g., logo.png) to avoid changing HTML paths, or update the HTML src attributes to match your new files.

7. Contact Form Functionality

The template comes with a fully functional contact form pre-configured to send emails without any backend code, using FormSubmit.co via AJAX.

How it Works

  • No Page Reload: The form submits asynchronously (AJAX), offering a smooth user experience.
  • Instant Feedback: The button changes to "Sending...", and a success message appears inline upon completion.
  • Spam Protection: The integration handles basic spam filtering automatically.
‚ö†Ô∏è CRITICAL: First-Time Activation
When you (or a visitor) submit the form for the very first time, FormSubmit will send an Activation Email to your-email@example.com.

You MUST click the "Activate Form" button in that email. Once activated, all future submissions will land directly in your inbox instantly.

Changing the Recipient Email

To send inquiries to a different email address:

  1. Open script.js in your code editor.
  2. Locate the fetch URL (around line 98): 
    fetch("https://formsubmit.co/ajax/YOUR_EMAIL@gmail.com", { ...
  3. Replace contact@example.com with your preferred email address.
  4. Submit the form once yourself to trigger the new activation email.
