Mobintix Infotech - Themeforest Submission Package

This directory is organized for upload to Themeforest/Envato.

1. /Template
   Contains the full source code (HTML, CSS, JS) of the template.
   This is the "Main File" that customers will use.

2. /documentation
   Contains the help file (index.html) and assets.
   Refers to its own copy of the preview image.

3. Missing Assets
   Please add the following images to this root directory before zipping, or prepare them for the upload form:
   - thumbnail.png (80x80 px) - Required for the marketplace listing.
   - theme_preview.png (approx 590x300 px) - Required for the item preview.

Instructions:
- Compress (Zip) the 'Template' and 'documentation' folders into a single file (e.g., Mobintix_Theme.zip) for the "Main File" upload.
- Upload 'thumbnail.png' separately as the "Thumbnail".
- Upload 'theme_preview.png' separately as the "Theme Preview".

Note: Ensure you check the specific category requirements on Themeforest as they can change.
